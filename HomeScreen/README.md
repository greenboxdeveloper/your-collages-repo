# Home screen configuration

## Files

| File | Role |
|------|------|
| `home_screen_blueprint.json` | **Edit this** — section order after the fixed header, titles, and slot types |
| `home_config.json` | **Generated by CI** — do not edit by hand |

Bundled fallback: `PhotoCollageMaker/PhotoCollage/Resource/home_config.json` (same content).

## Fixed header (always first)

These six sections are always prepended in this order (override titles/counts via `fixed_prefix` in the blueprint if needed):

1. `slideshow`
2. `tools_grid`
3. `template_store_row`
4. `popular_layouts`
5. `continue_projects`
6. `made_for_you`

## Tail sections (`sections` in blueprint)

Use `slot` to define dynamic rows. Examples:

| Slot | Purpose |
|------|---------|
| `template_row` / `hero_template` | One template category (`sub_category` fuzzy-matched to `Templates/`) |
| `template_rows` | One row per template folder (`max_rows`, `count`) |
| `filter_row` | Filter items row for one category |
| `filter_rows` | Multiple filter categories (`exclude`: `["basic"]`) |
| `sticker_row` / `sticker_rows` | Sticker packs |
| `background_row` / `background_rows` | Background packs |
| `fonts_row` | Font category (`pick`: `first` or set `sub_category`) |
| `classic_layouts` / `stylish_layouts` | Layout carousels from `enhanced_manifest.json` |

`sub_category` values are resolved against generated manifests; typos are corrected when close enough.

## Generation (CI)

Runs at the end of `scripts/generate_enhanced_manifest.py` when `--generate-home-config` is set:

- Reads manifests: `Filters/filter_manifest.json`, `Stickers/…`, `Backgrounds/…`, `Fonts/…`, `Templates/templates_index.json`
- Writes `home_config.json` and bumps **`version`** every run (e.g. `1.2` → `1.3`)
- Validates output in the same process (fails CI if invalid)

## Local test

After store manifests exist in the repo:

```bash
python scripts/generate_enhanced_manifest.py \
  --base-url "https://raw.githubusercontent.com/OWNER/REPO/main" \
  --repo-root . \
  --generate-filter-manifest \
  --generate-store-manifests \
  --generate-templates-index \
  --generate-home-config
```

Use bundled manifests for a quick check:

```bash
python scripts/generate_enhanced_manifest.py \
  --base-url "https://example.com/main" \
  --repo-root . \
  --filter-output PhotoCollageMaker/PhotoCollage/Resource/filter_manifest.json \
  --stickers-output PhotoCollageMaker/PhotoCollage/Resource/shape_store_manifest.json \
  --backgrounds-output PhotoCollageMaker/PhotoCollage/Resource/background_store_manifest.json \
  --fonts-output PhotoCollageMaker/PhotoCollage/Resource/font_catalog.json \
  --templates-index-output PhotoCollageMaker/PhotoCollage/Templates/templates_index.json \
  --generate-home-config
```

(Adjust paths to match files present locally.)
